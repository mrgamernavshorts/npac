# Npac
A custom-made aur helper written in c!

_(Sorry for the HORRIBLY written code. I'm just learning C :D)_

## Installation

If you already have a aur helper like `yay` or `paru`,

for yay:

```bash
yay -S npac-git
```

for paru:

```bash
paru -S npac-git
```

Use these commands:

```bash
# Clone the Repo
git clone https://github.com/mrgamernavshorts/npac

# Cd into npac repo
cd npac

#build
make npac

#move it to /usr/bin/ for systemwide access
sudo cp npac /user/bin/npac
```

## Removal

To remove npac, use this command:

```bash
#Remove npac from /usr/bin
sudo rm /usr/bin/npac
```

## Useage

Install the program and type:

```bash
npac --help
```
